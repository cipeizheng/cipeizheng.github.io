DIMM 基本原理与架构

DIMM 简介 DIMM（Dual Inline Memory Module，双列直插式内存模块），简单来说，就是PC里插的内存条，观察内存条，他的接口两侧都有金手指，这就是“双列”的含义。在内存条上，有很多方形的小芯片，这些芯片就是实际存储数据的DRAM芯片。 DIMM的设计允许每个引脚在模块的两侧都有连接，这样可以提供更高的引脚数量和更大的数据传输带宽（相较于SIMM，单列直插式内存模块）。接下来我们介绍一下 DIMM 的层级结构。 DIMM 层级结构 DIMM 的层级结构可以分为以下几个层次： DIMM：这是最顶层的结构，包含多个 DRAM 芯片。DIMM 模块通过金手指与主板上的内存插槽连接。 Rank：DIMM 可以包含一个或多个 Rank。每个 Rank 是一组 DRAM 芯片，通常是 8 个或 16 个芯片，它们共同工作以提供一个数据总线宽度（一般来说是 64 位，如果是 ECC 内存，则是 72 位）。每个 Rank 可以独立地进行读写操作。但是，内存控制器在同一时间只能访问一个 Rank。 Chip：每个 Rank 由多个 DRAM 芯片组成。每个芯片通常有一个数据宽度、一般分为x4、x8、x16等类型。x8 芯片表示每个芯片有 8 位数据宽度，那么 8 个 x8 芯片就可以组成一个 64 位的数据总线，也即一个 Rank。 Bank Group：每个 DRAM 芯片内部通常包含多个 Bank Group。Bank Group 是 DRAM 内部的一个逻辑划分，用于提高并行访问能力。每个 Bank Group 包含多个 Bank。DDR4 通常为 2 或 4 个 Bank Group（每个含 4 个 Bank，共 8/16 个 Bank）；DDR5 扩展到 8 个 Bank Group（每个含 4 个 Bank，共 32 个 Bank） ...
