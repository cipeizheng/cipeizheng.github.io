一、最建议日常关注的 LLM 榜单 #

榜单直接打开主要测试内容适合用来看什么
LMArena / Arenaarena.ai/leaderboard真实用户匿名对战投票，覆盖 text、image、vision 等模型对比。看“实际聊天体验”和用户偏好，不适合当严格客观题分数。(Arena AI)
Artificial Analysis LLM Leaderboardartificialanalysis.ai/leaderboards/models综合比较 intelligence、价格、输出速度、首 token 延迟、上下文窗口等。选 API、看性价比、速度和能力平衡。(人工分析)
LiveBenchlivebench.ai每月更新题目，覆盖数学、代码、推理、语言、指令遵循、数据分析；用客观答案自动评分，降低污染。看较新的通用硬能力，比老 MMLU/GSM8K 更有参考价值。(LiveBench)
OpenCompass / CompassRankrank.opencompass.org.cn多能力维度评测大语言模型和多模态模型，覆盖开源/闭源、中文/英文模型。看国产模型、开源模型、中文能力、综合能力。(OpenCompass)
HELM Capabilities / Safetycrfm.stanford.edu/helmStanford CRFM 的透明、可复现评测框架；包含能力、安全、视觉语言等榜单。看更学术、更重视评测透明性的结果。(斯坦福CRFM)
Scale AI SEAL Leaderboardslabs.scale.com/leaderboard专家驱动/私有数据集评测，覆盖 coding、reasoning 等方向。看更难被刷榜的数据集，但要注意它是商业机构维护的榜单。(Scale Labs)
Vellum LLM Leaderboardvellum.ai/llm-leaderboard汇总 2024 年 4 月之后发布的 SOTA 模型公开 benchmark 结果，并排除部分饱和老 benchmark。快速看新模型横向对比，适合做入口页。(Vellum)

二、代码 / 软件工程能力榜单 #

榜单直接打开主要测试内容适合用来看什么
SWE-bench Verifiedswebench.com/verified.html给真实 GitHub issue 和代码库，让模型/agent 生成 patch，再用测试判断是否修复；Verified 子集包含人工验证的 500 个样本。看真实软件工程修 bug 能力；注意它测的是“模型 + agent scaffold”。(SWE-bench)
SWE-bench 总榜swebench.comSWE-bench、SWE-bench Lite、Verified、Multilingual 等多个软件工程榜单。看 coding agent 生态整体进展。(SWE-bench)
LiveCodeBenchlivecodebench.github.io/leaderboard.html持续收集新编程题，覆盖代码生成、自修复、代码执行、测试输出预测等能力。看算法题/代码推理能力，比 HumanEval、MBPP 更抗污染。(LiveCodeBench)
Terminal-Benchtbench.ai/leaderboard让 agent 在真实终端环境中完成软件工程、机器学习、安全、数据科学等任务。看 CLI coding agent、自动跑命令、改项目、配置环境能力。(Terminal-Bench)
Terminal-Bench 2.1tbench.ai/leaderboard/terminal-bench/2.1Terminal-Bench 2.0 的修订版，修复部分任务并引入更严格验证。看最新终端 agent 榜。(Terminal-Bench)

三、Agent / 工具调用 / 浏览器操作榜单 #

榜单直接打开主要测试内容适合用来看什么
BFCL / Berkeley Function Calling Leaderboardgorilla.cs.berkeley.edu/leaderboard.html函数调用/工具调用准确性，包括串行、并行、多语言、REST/SQL 等形式。看模型是否适合做 tool calling、MCP、LangGraph agent。(Gorilla)
GAIA Leaderboardhuggingface.co/spaces/gaia-benchmark/leaderboard真实世界问题，需要推理、多模态、浏览网页、工具使用等能力。看“通用智能助手 / Deep Research 类 agent”能力。(Hugging Face)
WebArena Leaderboardleaderboard.steel.dev/leaderboards/webarena浏览器 agent 在购物、论坛、GitLab、CMS、地图、Wiki 等自托管网站上完成 812 个任务。看网页操作 agent 能力。(Steel.dev)
BrowserGym Leaderboardhuggingface.co/spaces/ServiceNow/browsergym-leaderboardWeb navigation agent 榜单；BrowserGym 集成 MiniWoB、WebArena、VisualWebArena、WorkArena、AssistantBench、WebLINX 等。看浏览器自动化 agent 的综合能力。(Hugging Face)
τ-bench / Tau-benchtaubench.com模拟真实客服场景，让 agent 与用户协作并使用工具完成任务。看多轮对话 + 工具调用 + 遵循业务规则能力。(τ-bench)
τ²-Bench / Tau-2 Benchartificialanalysis.ai/evaluations/tau2-bench双控制 conversational AI benchmark，模拟 telecom 技术支持场景。看客服 agent、企业流程 agent 能力。(人工分析)

四、中文大模型榜单 #

榜单直接打开主要测试内容适合用来看什么
SuperCLUEsuperclueai.com中文通用大模型综合测评，覆盖通用、文本、多模态、Agent 等能力。看中文综合能力和国产模型横评。(SuperCLUE)
C-Eval Leaderboardcevalbenchmark.com/static/leaderboard.html13,948 道中文多选题，覆盖 52 个学科、四个难度层级，并有 C-Eval Hard。看中文考试型知识/推理能力。(Ceval Benchmark)
CMMLUgithub.com/haonan-li/CMMLU中文语境下的知识和推理评测，覆盖 67 个主题，包括自然科学、人文社科、工程等。看中文文化/中国语境知识能力。(GitHub)
AlignBenchgithub.com/THUDM/AlignBench中文 alignment 评测，覆盖基础语言、高级中文理解、开放问答、写作、逻辑、数学、角色扮演、专业知识等。看中文指令跟随、开放式回答质量和对齐效果。(GitHub)

五、长上下文 / RAG / Embedding 相关榜单 #

榜单直接打开主要测试内容适合用来看什么
LongBench v2longbench2.github.io长上下文理解与推理，文本长度从 8K 到 2M words，多数在 128K 以下。看模型是否真的能读长文档、做长上下文推理。(LongBench)
MTEB Leaderboardhuggingface.co/spaces/mteb/leaderboard文本 embedding 模型评测，覆盖检索、分类、聚类、重排序等任务；MTEB 原始论文覆盖 8 类任务、58 个数据集、112 种语言。做 RAG、向量检索、语义搜索时选 embedding 模型。(Hugging Face)
MMNeedlemmneedle.github.io多模态 needle-in-a-haystack，测试模型在大量图片/视觉上下文中定位目标信息。看多模态长上下文能力。(MM Needle)

六、Agent memory / 长期记忆方向 #

榜单/项目直接打开主要测试内容适合用来看什么
LongMemEvalxiaowu0162.github.io/long-mem-eval长期对话记忆评测，测试 chat assistant 是否能跨大量历史会话回忆、推理。看长期记忆系统、memory layer、RAG memory。(Di Wu)
LongMemEval-V2github.com/xiaowu0162/LongMemEval-V2长期 agent memory 评测，包含 evaluation harness、数据准备工具、leaderboard packaging utilities。看更新的 agent memory benchmark。(GitHub)
LoCoMosnap-research.github.io/locomo超长期对话记忆，每个对话平均 300 turns、9K tokens、最多 35 个 session；覆盖 QA、事件摘要、多模态对话生成。看多 session 个人记忆、时间推理、跨会话召回。(Snap Research)
STATE-Benchgithub.com/microsoft/STATE-BenchMicrosoft 的 agent memory benchmark，评测 agent 是否能在旅行、客服、购物等企业流程中随经验改进。看“经验积累型 agent / enterprise memory”。(GitHub)

建议实际收藏顺序 #

你如果只是日常跟踪 LLM 发展，先收藏这 8 个就够：

  1. LMArena / Arena
  2. Artificial Analysis
  3. LiveBench
  4. SWE-bench Verified
  5. LiveCodeBench
  6. BFCL
  7. Terminal-Bench
  8. OpenCompass
